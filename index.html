scratch

Scratch - это визуальный язык программирования, созданный для обучения детей и начинающих программистов основам кодирования. В отличие от текстовых языков программирования, где команды записываются словами и символами, в Scratch программы создаются путем перетаскивания блоков команд и сборки их в цепочки.

Основные элементы Scratch:

Сцена: Место, где располагаются спрайты (объекты) и где происходит вся программа.

Спрайты: Графические объекты, которые можно анимировать и управлять в рамках программы.

Блоки: Это команды, которые пользователь перетаскивает на сцену и собирает вместе, чтобы создать программу. Блоки представлены в виде разноцветных форм, каждая из которых выполняет определенное действие (например, перемещение спрайта, изменение цвета и т. д.).

Программирование на Scratch обычно начинается с создания простых анимаций или интерактивных игр. Пользователи могут управлять движением спрайтов, изменять их внешний вид и создавать условия для реакции на действия пользователя или других спрайтов.

Начинающие программисты на Scratch учатся основным концепциям программирования, таким как последовательность действий, циклы, условия и события. Это позволяет им развивать свои навыки и переходить к более сложным проектам по мере увеличения опыта.

HTML

Вот краткий обзор некоторых HTML тегов и их функций:

p - Этот тег используется для создания абзаца текста. Он помогает организовать контент на веб-странице, разделяя его на логические блоки текста.

a - Тег для создания гиперссылок. Он позволяет создавать ссылки на другие веб-страницы, изображения, файлы и т.д.

img - С помощью этого тега вставляют изображения на веб-страницу. Он использует атрибуты, такие как src (для указания пути к изображению) и alt (для текстового описания изображения).

ul и ol - Эти теги используются для создания неупорядоченных и упорядоченных списков соответственно. Внутри них обычно используется тег li для создания элементов списка.

div - Это блочный элемент, который используется для группировки других элементов HTML вместе. Он часто применяется для создания различных секций и макетов на веб-странице.

span - Этот тег, в отличие от div, является строчным элементом и используется для выделения или стилизации отдельных фрагментов текста или других элементов внутри строки.

input - Тег для создания элементов ввода на веб-странице, таких как текстовые поля, кнопки, флажки и т.д.

table - Этот тег используется для создания таблиц на веб-странице. Внутри него обычно используются теги tr для строк таблицы, td для ячеек данных и th для заголовков.

form - Тег, который обозначает форму для ввода данных пользователем. Он может содержать различные элементы ввода, такие как текстовые поля, флажки, выпадающие списки и кнопки отправки.

iframe - Этот тег используется для встраивания веб-страниц или другого контента внутрь текущей веб-страницы.

Это только небольшой обзор некоторых HTML тегов и их основных функций. HTML предоставляет широкий набор тегов для создания различных элементов и структур на веб-страницах.

CSS стили

color: Задает цвет текста элемента.
font-size: Устанавливает размер шрифта.
font-family: Устанавливает семейство шрифтов для текста.
font-weight: Устанавливает насыщенность шрифта (например, нормальный, жирный).
text-decoration: Определяет стиль декоративных линий текста (например, подчеркивание, перечеркивание).
background-color: Задает цвет фона элемента.
border: Задает границу элемента.
box-shadow: Создает тень вокруг элемента.
text-align: Выравнивает текст внутри элемента по горизонтали.
line-height: Устанавливает высоту строки в тексте.

2022

CSS структурирование

display: Определяет тип отображения элемента (например, блочный, строчный, инлайн-блок).
position: Управляет позиционированием элемента на веб-странице (например, абсолютное, относительное, фиксированное).
float: Определяет, как элемент выравнивается по сторонам (например, влево или вправо) относительно окружающих элементов.
clear: Указывает, какие стороны элемента должны быть очищены от обтекающих элементов.
flexbox: Позволяет легко распределить элементы внутри контейнера с помощью гибкого макета.
grid: Предоставляет мощный и гибкий способ создания сеточных макетов для веб-страниц.
margin: Устанавливает внешние отступы вокруг элемента.
padding: Устанавливает внутренние отступы вокруг содержимого элемента.
overflow: Управляет тем, как браузер обрабатывает содержимое элемента, когда оно не помещается в пределах его размеров.
z-index: Устанавливает порядок наложения элементов, определяя, какие элементы находятся перед или за другими.

CSS анимации

@keyframes: Определяет анимацию путем изменения стилей во времени.
animation: Применяет анимацию к элементу и управляет ее параметрами.
transition: Задает плавные переходы между состояниями элемента.
transform: Применяет преобразования к элементу, такие как перемещение, поворот и масштабирование.
opacity: Устанавливает прозрачность элемента.
rotate: Поворачивает элемент на определенный угол.
scale: Масштабирует размер элемента.
translate: Перемещает элемент на определенное расстояние.
skew: Наклоняет элемент на заданный угол.
animation-delay: Задает задержку перед запуском анимации.
